# sql-sales-analysis
SQL project analyzing sales data to extract business insights like top customers, regions, and delivery performance.
Tools Used
- MySQL
- SQL
Key Insights
- Identified top customers by sales
- Analyzed region-wise sales performance
- Found top-performing products
- Calculated average delivery time

# Key Learning
Handled data cleaning by converting text-based date columns using STR_TO_DATE for accurate analysis.

# Conclusion
This project demonstrates SQL skills including aggregation, grouping, sorting, and data cleaning.
